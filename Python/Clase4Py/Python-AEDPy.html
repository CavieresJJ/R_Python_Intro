Anlisis Exploratorio de Datos con Python

Se realizar un anlisis exploratorio de uno de los ms famosos datasets, el IRIS, con informacin sobre 3 tipos de plantas. Este dataset es comnmente usado en problemas de clasificacin, cuando el objetivo es prever la clase de los datos. En este caso, prever la categora de una planta a partir de medidas de la planta (sepal e petal).

Dataset (ya disponible con el Scikit-Learn): https://archive.ics.uci.edu/ml/datasets/iris
